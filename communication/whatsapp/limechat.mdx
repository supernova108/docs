This guide provides the steps to integrate LimeChat with ReferRush for WhatsApp messaging.

## Part 1: Create a Campaign (Template) in LimeChat

Follow these steps within your LimeChat dashboard.

<Steps>
  <Step>
    ### Step 1: Create a Template

    *   Create a new Template in LimeChat.
    *   Use the message text provided by ReferRush.
    *   Set the Call-to-Action (CTA) button to dynamic and keep it as default.

  </Step>

  <Step>
    ### Step 2: Set Up New Automation

    *   Create a new automation flow.
    *   Choose the trigger as given in the ReferRush template preview page.
    *   Add the template to the flow.
    *   Add the variables to the flow.
    *   Add the header image to the flow.
    *   Launch the automation.

  </Step>

  <Step>
    ### Step 3: Setup Automation Variables

    *   Set up the necessary automation variables as given in the ReferRush template preview page.

  </Step>

  <Step>
    ### Step 4: Gather Required Information

    *   Note the Template ID of the created template.
    *   Find your API Key and Account ID from your LimeChat account settings.

    <Info>
      You will need the `Template ID`, `API Key`, and `Account ID` from LimeChat for the next part.
    </Info>

  </Step>
</Steps>

## Part 2: Configure ReferRush

Gather the required information from LimeChat and enter it into your ReferRush settings.

<Steps>
  <Step>
    ### Step 1: Locate Template ID in LimeChat

    1.  Log in to your LimeChat dashboard.
    2.  Navigate to the "Templates" section.
    3.  Find the template for the specific event type.
    4.  Copy the **Template ID** from the template details.

  </Step>

  <Step>
    ### Step 2: Find API Key and Account ID in LimeChat

    *   Find your **API Key** and **Account ID** within your LimeChat account settings. Copy them.
  </Step>

  <Step>
    ### Step 3: Enter Details in ReferRush

    *   Navigate to the configuration section for the specific message template you are setting up.
    *   Paste the **Template ID** (copied in Step 1) into the "Template ID" field.
    *   Paste your **API Key** and **Account ID** (copied in Step 2) into the corresponding fields.
    *   Save the settings in ReferRush.

  </Step>
</Steps>

## Part 3: Test the Integration
<Steps>
  <Step>
    ### Step 1: Test the Integration
    *   Go to the test tab
    *   Test the integration by sending a test message from ReferRush.
  </Step>
</Steps>

If all the things are working fine, the setup using the provided steps is now complete click on Finish.
ReferRush will use the Template ID, API Key, and Account ID to send messages via LimeChat.
