This guide provides the steps to integrate SagePilot with ReferRush for WhatsApp messaging.

## Part 1: Create a Campaign (Template) in SagePilot

Follow these steps within your SagePilot dashboard.

<Steps>
  <Step>
    ### Step 1: Create a Template

    *   Create a new Template in SagePilot.
    *   Use the message text provided by ReferRush.
    *   Set the Call-to-Action (CTA) button to dynamic and keep it as default.

  </Step>

  <Step>
    ### Step 2: Set Up New Automation

    *   Create a new automation flow.
    *   Choose the trigger as given in the ReferRush template preview page.
    *   Add the template to the flow.
    *   Add the variables to the flow.
    *   Add the header image to the flow.
    *   Launch the automation.

  </Step>

  <Step>
    ### Step 3: Setup Automation Variables

    *   Set up the necessary automation variables as mentioned in the ReferRush template preview page.

  </Step>

  <Step>
    ### Step 4: Get Configuration Parameters

    *   Contact the SagePilot team to get the required parameters for the configuration section in ReferRush.

    <Info>
      You will need the parameters provided by SagePilot and the template name for the next part.
    </Info>

  </Step>
</Steps>

## Part 2: Configure ReferRush

Gather the required information and enter it into your ReferRush settings.

<Steps>
  <Step>
    ### Step 1: Get Parameters from SagePilot

    *   Contact the SagePilot team to obtain the necessary parameters for integration.

  </Step>

  <Step>
    ### Step 2: Enter Details in ReferRush

    *   Navigate to the configuration section for the specific message template you are setting up.
    *   Enter the parameters provided by SagePilot into the corresponding fields.
    *   Enter the template name in the "Template ID" field.
    *   Save the settings in ReferRush.

  </Step>
</Steps>

## Part 3: Test the Integration
<Steps>
  <Step>
    ### Step 1: Test the Integration
    *   Go to the test tab
    *   Test the integration by sending a test message from ReferRush.
  </Step>
</Steps>

If all the things are working fine, the setup using the provided steps is now complete click on Finish.
ReferRush will use the provided parameters and the template name to send messages via SagePilot.
